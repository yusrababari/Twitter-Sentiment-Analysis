## Signal — Tweet Sentiment Reader 📡

Signal is a web application that analyzes and visualizes the sentiment of tweets. It uses an external sentiment analysis API to classify tweet text into emotional categories while providing a modern, minimalist interface inspired by developer tools and terminal design systems.

🚀 **Live Demo:** https://twitter-sentiment-analysis-sepia.vercel.app/

## ✨ Features

- **AI-Powered Sentiment Analysis:** Uses an API integration to analyze tweet text and classify sentiment as Positive, Negative, or Neutral.
- **Real-Time Results:** Sends user input to the sentiment analysis API and displays instant sentiment predictions.
- **Clean UI / UX:** High-contrast, developer-focused aesthetic inspired by modern terminal interfaces and design systems.
- **Fast Performance:** Built with Vite and React for fast rendering and a lightweight application bundle.

## 🛠️ Tech Stack

- **Frontend Framework:** React
- **Build Tool:** Vite
- **Styling & Fonts:** CSS3, Space Grotesk, IBM Plex Mono
- **API Integration:** Sentiment Analysis API
- **Deployment:** Vercel
