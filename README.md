# Signal — Tweet Sentiment Reader 📡

**Signal** is a web application that analyzes and visualizes sentiment in tweets. Built with a modern, minimalist UI using **Space Grotesk** and **IBM Plex Mono** typography.

🚀 **Live Demo:** twitter-sentiment-analysis-sepia.vercel.app
---

## ✨ Features

- **Sentiment Analysis:** Analyzes tweet text to determine emotional tone and sentiment (Positive, Negative, Neutral).
- **Clean UI / UX:** High-contrast, clean developer-first aesthetic inspired by modern terminal & design design systems.
- **Fast Performance:** Powered by Vite and React for instantaneous rendering and lightweight bundle size.

---

## 🛠️ Tech Stack

- **Frontend Framework:** [React](https://react.js.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling & Fonts:** CSS3, [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk), [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono)
- **Deployment:** [Vercel](https://vercel.com/)
